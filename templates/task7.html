📘 Supervised Learning Explanation

Supervised learning is a machine learning technique where the model is trained using labeled data. The algorithm learns relationships between input variables and output variables and then predicts results for new data.

⚙️ Dataset Splitting

Training Dataset

Used to train the machine learning model and help it learn patterns from the dataset.

Testing Dataset

Used to evaluate how well the trained model performs on unseen data.

⚠️ Overfitting vs Underfitting

Overfitting

The model memorizes the training data instead of learning patterns, resulting in poor performance on new data.

Underfitting

The model is too simple and cannot capture the relationships between input and output variables.
