🎯 Student Performance Prediction Model

This module uses machine learning to predict student exam performance based on study habits, attendance, sleep patterns and motivation level. The system analyzes these factors and predicts the expected exam score.

📚 Model Description

The prediction model is built using regression techniques from machine learning. The algorithm studies the relationship between multiple factors affecting academic performance such as study hours, attendance, sleep hours, previous scores and motivation level.

By analyzing these variables, the system predicts the student's expected exam score and performance category.

⚙ Factors Used In Prediction

  • Study Hours
  • Attendance Percentage
  • Sleep Hours
  • Previous Academic Scores
  • Motivation Level
  • Internet Access
  • Extracurricular Activities
