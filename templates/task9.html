🧠 Understanding Overfitting and Underfitting

📉 Underfitting

Underfitting occurs when a machine learning model is too simple to capture the underlying patterns of the dataset. In this situation the model fails to learn the relationship between independent variables and the target variable.

  • High Training Error
  • High Testing Error
  • Model too simple
  • Important patterns not captured

📈 Overfitting

Overfitting happens when the model becomes too complex and starts memorizing the training dataset instead of learning general patterns. This causes the model to perform extremely well on training data but poorly on unseen data.

  • Very Low Training Error
  • High Testing Error
  • Model memorizes noise
  • Poor generalization

⚖️ Balanced Model

A well trained model balances bias and variance. It captures patterns from the dataset without memorizing noise and therefore performs well on both training and testing datasets.

  • Low Training Error
  • Low Testing Error
  • Optimal complexity
  • Good prediction accuracy

⚙️ Effect of Model Complexity

Model complexity refers to how flexible the machine learning model is in capturing relationships in the data. Increasing complexity allows the model to fit training data more closely, but excessive complexity leads to overfitting.

In this experiment we used polynomial regression with different degrees to demonstrate how training and testing errors behave with increasing model complexity.
