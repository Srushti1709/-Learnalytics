📘 About This Module

This feature allows users to analyze student academic performance using machine learning techniques. Users can either manually input student data or upload a dataset for analysis.

The system processes input data and applies predictive models to estimate student performance based on factors such as study hours, attendance, sleep, and motivation.

This helps in identifying key patterns and making data-driven academic decisions.
